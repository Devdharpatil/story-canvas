# Pocket Writer

A versatile content creation platform that combines the intuitive design experience of Canva with the clean reading flow of Medium.

## Project Structure

- **story-canvas-android**: Android application built using Jetpack Compose
- **story-canvas-backend**: Spring Boot backend with Kotlin

## Features

- **Feed View**: Browse articles in a clean, Medium-inspired interface
- **Template Builder**: Create custom layouts with drag-and-drop functionality
- **Article Creation**: Create and publish visually rich content with ease

## Setup Instructions

### Backend (Spring Boot)

1. Navigate to the `story-canvas-backend` directory
2. Run `./gradlew bootRun` to start the Spring Boot application

### Android App

1. Navigate to the `story-canvas-android` directory
2. Open the project in Android Studio
3. Run the app on an emulator or physical device
