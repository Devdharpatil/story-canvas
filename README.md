# Pocket Writer

An application for writers to create, organize, and manage their stories. The project consists of:

1. **Android Mobile App** - For creating and managing stories on mobile devices
2. **Backend Server** - Spring Boot based REST API to manage story data 

## Project Structure

- `StoryCanvas/story-canvas-android/` - Android mobile application
- `StoryCanvas/story-canvas-backend/` - Spring Boot backend server
- `run-server.bat` - Convenience script to start the backend server

## Setup Instructions

### Backend Server

1. Ensure you have Java JDK 17+ installed
2. Navigate to the `story-canvas-backend` directory
3. Run `./gradlew bootRun` or use the provided `run-server.bat` file

### Android Application

1. Open the `story-canvas-android` directory in Android Studio
2. Sync Gradle project
3. Run the application on an emulator or physical device
